Some JINJA2 Exercises/Demos:

    Iterate through a list and use a conditional:

    {% for fruit in my_fruit_list %} {% if fruit == "limes" %}
  • {{fruit}}
    • {% else %}
  • {{fruit}}
    • {% endif %} {% endfor %}

Iterate through a list of dictionaries:

{% for person in my_person_list %}
Name: {{person.name}}
Age: {{person.age}}
{% endfor %}

Macro Demo:

{% for person in my_person_list %} {{ my_macro(person.name, person.age) }} {% endfor %}

Sort By Name:

{# https://jinja.palletsprojects.com/en/2.11.x/templates/#builtin-filters #} {% for person in my_person_list | sort(attribute="name") %} {{ my_macro(person.name, person.age) }} {% endfor %}

Assigning variables and comparing strings and ints in Jinja:

{% set my_string = '5' %} {% set my_int = 5 %} my_string: {{ my_string }}
my_int: {{ my_int }}
{% if my_string == my_int %} my_string and my_int are equal
{% else %} my_string and my_int are not equal but . . . .
{% endif %} {% if my_string == my_int | string %} my_string and "my_int | string" are equal
{% else %} my_string and my_int are not equal
{% endif %} {% endblock %}
